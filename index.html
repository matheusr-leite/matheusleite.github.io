Sobre o Biopark

O Biopark é um legado, que vem sendo deixado para a região de Toledo. Foi lançado oficialmente em 2016, porém já vem sendo idealizado por seus fundadores, Carmem e Luiz Donaduzzi, desde os anos 2000. É um Parque tecnológico, com mais de 5 milhões de m², que reune educação, pesquisa e negócios. Clique aqui para saber mais sobre o território da inovação e das oportunidades.

Uma das inúmeras iniciativas do Biopark é o Biopark Educação, seu braço educacional, que já conta com quatro cursos aprovados pelo mec, cursos de curta duração, projetos para formação de programadores como o "Biopark Connect" e "Do Zero ao Um". E todos tem como objetivo em comum, de colocar mais dinheiro no bolso de quem participa, como diz Donaduzzi. Lançando vestibulares com promessas muito desafiadoras, como: "Eu garanto seu trabalho, ou você estuda de graça.", utilizado na campanha do vestibular de 2022.

Portugol

Tivemos uma introdução a algoritimos utilizando a pseudolinguagem Portugol, que tem como objetivo eliminar a dificuldade liguistica no começo do desenvolvimento de algoritimos estruturados. Foi um momento importante para começar com os algoritimos, já que a lingua era acessível, e trouxe um fácil entendimento do que o código estava fazendo, o que ajudou na migração para o Java.

HTML, CSS E Java Script

Em programação front-end tivemos introdução ao HTML, CSS e JavaScript. O que já nos deu os poderes de criar interações com o usuário. Depois disso já começamos a estudar bootstrap, que ajuda demais na questão de estilização da nossa aplicação web.

C

Começamos ver a linguagem C na matéria de estrutura de dados, pois ela é a mais adequada para o assunto, aprendemos a usar vetores, matrizes, struct, a criar arquivos e salvar eles, e até mesmo criar um programa com persistência nos dados, um sistema para venda de passagens de ônibus.

Java

Utilizamos Java para continuar no assunto de algoritimos, depois de algum tempo utilizando Portugol. Apreendemos a sintaxe da linguagem, e finalizamos algoritimos e lógica de programação utilizando essa linguagem.

Engenharia de Software

Em engenharia de software de software fomos mais para conteúdos na área de atuação de analista, como engenharia de requisitos, criação das telas de um app e também escrevemos a documentação que seria necessária para o desenvolvimento desse aplicativo.

Metodologias Ágeis

Fomos apresentados, e trabalhamos com metodologias ágeis, como kanbam, scrum, scrumbam. Fizemos algumas sprints, com reuniões rápidas no começo do trabalhalho, fizemos discuções de encerramento para saber onde evoluir para a próxima sprint, demos pontos para as atividades que deveríamos realizar, para medir o que poderia ser feito dentro do período que estabelecemos.
